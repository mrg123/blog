# phpmailer[^1]

## hotmail smtp设置

hotmail smtp设置

接收 (POP3) 服务器

服务器地址：pop-mail.outlook.com

端口：995

加密连接：SSL

发送 (SMTP) 服务器

服务器地址：smtp-mail.outlook.com

端口：25（如果阻止了 25，则使用 587）

身份验证：是

加密连接：TLS

## gmail smtp 设置

接收邮件（POP3）服务器：pop.gmail.com

需要 SSL

端口：995



发送邮件（SMTP）服务器：smtp.gmail.com

需要 TLS 或 SSL

使用身份验证：是

TLS/STARTTLS 端口：587

SSL 端口：465



帐户名：您的完整电子邮件地址（包括 @gmail.com 或 @buicaa.com）

密码：您的 Gmail 密码
